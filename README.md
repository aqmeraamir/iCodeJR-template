# iCodeJR-template
 Website template made for a coding contest

# Software
- SolidJS ![SolidJS Logo](https://www.solidjs.com/img/logo/without-wordmark/logo.jpg)

# Run
- Download all the code as a ZIP file
- Un-zip it
- run `npm install`
- run `npm run dev`
